# Code of Conduct

This project is committed to a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical issues, not personal attacks.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment, discrimination, or hateful language
- Personal attacks or intimidation
- Doxxing or sharing private information without consent

## Enforcement

Project maintainers may remove comments, commits, issues, or users that violate this policy.
