# Contributing

Thanks for contributing to Nashville Numbers Converter.

## Development Setup

```bash
# from an activated .venv
pip install -e .
```

## Run Tests

```bash
python -m pytest -q
```

## Pull Requests

- Keep changes scoped and focused.
- Add or update tests for behavior changes.
- Update docs (`README.md`, `AGENTS.md`, `ROADMAP.md`) when relevant.
- Ensure tests pass before opening a PR.
